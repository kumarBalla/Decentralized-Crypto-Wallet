# Decentralized Crypto Wallet (MetaMask Clone)

## Project Overview

This project is a **MetaMask-inspired decentralized crypto wallet** that allows users to securely manage digital assets and interact with blockchain-based services. The application provides essential wallet features such as **buying, selling, and swapping cryptocurrencies**, along with transaction management and secure authentication.

## Technologies Used

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Backend:** Spring Boot (Java)
* **Database:** Oracle Database
* **API Communication:** REST APIs
* **Version Control:** Git & GitHub

## Features

* User authentication and secure login
* Crypto wallet dashboard
* Buy cryptocurrency functionality
* Sell cryptocurrency functionality
* Swap crypto assets
* Transaction history tracking
* Secure backend APIs for wallet operations
* Responsive user interface

## Installation and Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/Decentralized-Crypto-Wallet.git
```

### 2. Run Backend (Spring Boot)

```
cd Meta-backend
mvn spring-boot:run
```

### 3. Run Frontend (React)

```
cd Metamask
npm install
npm start
```
